## Receive Form Messages by Email

The contact form is already marked up for Netlify Forms. To receive every visitor message by email:

1. Deploy the website to Netlify.
2. Open the site in Netlify.
3. Go to Forms.
4. Open Form notifications.
5. Add your email address as an email notification recipient.

When opened locally from `index.html`, the form shows a demo success message because local HTML files cannot send email by themselves.
