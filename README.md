# Rami Seghir Portfolio Website

This is a modern one-page professional portfolio website for Rami Seghir, focused on commercial operations, purchasing follow-up, industrial procurement, import documentation, and international trade.

## Files

- `index.html` - Main website structure and content.
- `style.css` - Responsive premium styling, animations, cards, timeline, and layout.
- `script.js` - Mobile navigation, scroll reveal animations, active navigation state, and demo form behavior.

When opened locally from `index.html`, the form shows a demo success message because local HTML files cannot send email by themselves.
